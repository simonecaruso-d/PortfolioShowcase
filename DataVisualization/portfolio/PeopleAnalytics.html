01 · Problem

HR data in hundreds of different Excel files

Every domain (cost, recruiting, turnover, learning, and so on) lived in a separate workbook owned by a different person. No shared definitions, no common date spine, no way to cross-reference headcount against cost against exits

Before

  • Fragmented Excel files per domain: cost in one workbook, recruiting in another, exit data in a third: each with its own column naming and time logic
  • No single employee identifier across sources: joining Oracle ERP to SharePoint surveys required manual VLOOKUP on email, rebuilt every month
  • NY cost forecast was a copy-paste exercise: Finance received a snapshot, not a live model with scenario inputs
  • Zero visibility into turnover prediction: exits were reported after the fact, never anticipated

After

  • Single Master Registry hub joins Oracle ERP, SharePoint cost, recruiting pipeline, exit surveys, and eNPS into one governed model on a shared email
  • Compliance is treated as a first-class citizen in the model and receives a dedicated tech/data approach
  • NY cost forecast is a live scenario model: planned hires, exits, raises, and contract transformations update the closing figure dynamically
  • 6-variable ML regression in DAX produces per-employee churn probability with R² and MAPE diagnostics updated on each refresh

04 · DAX

The logic, in code

160+ measures across 8+ domain groups. The standout: a full multiple linear regression embedded as a calculated column, with R² and MAPE diagnostics depending on per-row SSR/SST columns iterated with EARLIER

Chargeability measure
Ratio of hours logged on chargeable activities to total productive hours excluding Absence, capped to dates up to today so future forecast timesheets don't inflate the rate
DAX
 1VAR Chargeable   = {"Market", "Production - Other Team", "Production - Same Team"}
 2VAR Numerator    = CALCULATE(SUM('Timesheet (Minimal)'[Hours]), 'Timesheet (Minimal)'[Competency Date] <= TODAY() && 'Timesheet (Minimal)'[Category] IN Chargeable)
 3VAR Denominator  = CALCULATE(SUM('Timesheet (Minimal)'[Hours]), 'Timesheet (Minimal)'[Competency Date] <= TODAY() && 'Timesheet (Minimal)'[Category] <> "Absence")
 4RETURN = IF(Denominator IN {BLANK(), 0}, "Error", Numerator / Denominator)
Average Salary Gap measure
Pay gap between protected-category employees and the rest
DAX
 1VAR AverageNotProtectedRAL = CALCULATE(AVERAGE('Timesheet (Minimal)'[Internal Price]), Anagrafica[CatP] IN {"NO", BLANK()})
 2VAR AverageProtectedRAL    = CALCULATE(AVERAGE('Timesheet (Minimal)'[Internal Price]), Anagrafica[CatP] = "SI")
 3RETURN IF(AverageNotProtectedRAL IN {0, BLANK()}, BLANK(), (AverageProtectedRAL - AverageNotProtectedRAL) / AverageNotProtectedRAL)
Out measure
Count of leavers for the selected period. Dual-granularity logic lets the same measure drive both annual summary cards and monthly trend charts without duplication
DAX
 1VAR Anno       = SELECTEDVALUE('Calendar Table'[Date].[Year])
 2VAR Mese       = SELECTEDVALUE('Calendar Table'[Date].[MonthNo])
 3VAR FiltroAnno = FILTER(Anagrafica, Anagrafica[Status] = "Non attivo" && Anagrafica[Contract End Date].[Year] = Anno)
 4VAR FiltroMese = FILTER(Anagrafica, Anagrafica[Status] = "Non attivo" && Anagrafica[Contract End Date].[Year] = Anno && Anagrafica[Contract End Date].[MonthNo] = Mese)
 5VAR OutputAnno = CALCULATE(DISTINCTCOUNT(Anagrafica[Employee]), FiltroAnno)
 6VAR OutputMese = CALCULATE(DISTINCTCOUNT(Anagrafica[Employee]), FiltroMese)
 7RETURN IF(Mese = BLANK(), OutputAnno, OutputMese)
RSquared measure
Model fit for the turnover regression, basedo on the sums of per-row SSR (squared residuals vs prediction) and SST (squared deviations from mean actual)
DAX
 1VAR SSR = SUMX('Z - Regression on Negative Turnover', 'Z - Regression on Negative Turnover'[Z - SSR])
 2VAR SST = SUMX('Regression on Negative Turnover', 'Regression on Negative Turnover'[Z - SST])
 3RETURN 1 - DIVIDE(SSR, SST)
NPS measure
Net Promoter Score from exit interviews
DAX
 1VAR Detrattori = CALCULATE(DISTINCTCOUNT('Exit Interview'[Name]), 'Exit Interview'[Z - NPS] < 7 && 'Exit Interview'[Z - NPS] > 0)
 2VAR Difensori  = CALCULATE(DISTINCTCOUNT('Exit Interview'[Name]), 'Exit Interview'[Z - NPS] > 8)
 3RETURN Difensori - Detrattori
Is it the last price for this person? column · Prices History
Flags the most recent price record per employee. Used as a filter anchor in salary gap, RAL estimation, and cost measures to avoid double-counting historical contract tiers
DAX
 1VAR Employee = 'Storico Tariffe'[Employee]
 2VAR Filter   = FILTER(ALL('Storico Tariffe'), 'Storico Tariffe'[Employee] = Employee)
 3VAR MaxDate  = CALCULATE(MAX('Storico Tariffe'[Price Date]), Filter)
 4VAR Date     = 'Storico Tariffe'[Price Date]
 5RETURN IF(MaxDate = Date, "Y", "N")
